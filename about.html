i'm eesa. i'm 16 years old and i've been coding for a really long time. since i was like 8 i think, maybe younger, and scratch was my first coding language. i don't have an account though, because i used the offline editor.

anyways, soon after, i started to learn python. i remember that i would feel really smart whenever i'd copy the tutorials and there wouldn't be any errors when i ran the program. i still feel the same way today when i copy tutorials. enough about that though.

i got bored of python at one point and tried learning html. i remember being like 10 years old and making goofy "portfolio" websites with terrible css, but hey, let's be real, we've all been there at one point. my "portfolio" consisted of some of my other html projects. projects like "all about lego" and "super mario website" and stuff.

i got better with html and css over time through practice. my next goal was to learn javascript. i got some library books and began adding js to my sites. i remember how awesome it felt to finally understand how to handle forms, buttons, and inputs.

i started to learn react, and the syntax took a while to get used to at first. later on, i began to learn the "mern" stack. this is around when i started to learn typescript and tailwind as well.

once i was out of "tutorial hell" i decided to work on a hobby project: lasersocial - a fully functioning social media platform. i tell people that i built it to practice my full stack development skills, but at the time, my reason for making it was because my parents didn't allow me to use social media. i got a few of my friends to join but they really only used it for a few days.

when i entered high school, i kind of "lost interest" in coding for a while and dedicated more of my time towards some of my other hobbies. however, during the summer of 2024, i received an email from github's newsletter about hack club arcade. i spent a lot of my summer working on some projects for arcade, and i've been (somewhat) active in the hack club community ever since.

i code quite a lot nowadays (writing this in may 2025), check out my github and see what i'm working on. i'm gonna be honest, i should really be spending more of my time prepping for college applications.

thanks for reading. by the way if you're a hackclubber, DM me on slack about what you think of this website!
